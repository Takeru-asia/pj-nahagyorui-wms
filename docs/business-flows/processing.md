# 加工実績報告フロー（現状）

## 概要

製造部が加工作業を行い、その出来高を事務員へ報告するまでの業務フローを記載する。

**主な加工対象**: マグロ

---

## 業務フロー

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   製造部    │    │   事務員    │    │ 販売管理    │
│             │    │             │    │ システム    │
└──────┬──────┘    └──────┬──────┘    └──────┬──────┘
       │                  │                  │
       │ 1. 材料出庫      │                  │
       │   （マグロ）     │                  │
       │                  │                  │
       │ 2. 加工作業      │                  │
       │                  │                  │
       │ 3. 出来高記録    │                  │
       │   （手書き帳票） │                  │
       │                  │                  │
       │ 4. 帳票を        │                  │
       │    事務員へ      │                  │
       │─────────────────>│                  │
       │                  │                  │

       │                  │─────────────────>│
       │                  │                  │
```

## 報告内容

| 項目 | 内容 | 例 |
|------|------|-----|
| 材料出庫数 | 加工に使用した原材料の数量 | マグロ ○本 |
| 出来高 | 加工後の製品数量 | 500gブロック ○個、70gブロック ○個 |

---

## 各ステップの詳細

### 1. 材料出庫

| 項目 | 内容 |
|------|------|
| 担当者 | 製造部 |
| 場所 | 倉庫 → 加工場 |
| 手段 | マグロ等の原材料を倉庫から出庫 |
| 備考 | 出庫記録の有無は不明 |

### 2. 加工作業

| 項目 | 内容 |
|------|------|
| 担当者 | 製造部 |
| 場所 | 加工場 |
| 手段 | マグロを各サイズのブロック等に加工 |
| 備考 | - |

### 3. 出来高記録

| 項目 | 内容 |
|------|------|
| 担当者 | 製造部 |
| 場所 | 加工場 |
| 手段 | 手書きの帳票に記録 |
| 記録内容 | 材料出庫数、出来高（製品種類ごとの数量） |

### 4. 帳票の引き渡し

| 項目 | 内容 |
|------|------|
| 担当者 | 製造部 → 事務員 |
| 場所 | 社内 |
| 手段 | 手書き帳票の手渡し |
| 備考 | - |

### 5. システム入力

| 項目 | 内容 |
|------|------|
| 担当者 | 事務員 |
| 場所 | 社内 |
| 手段 | 販売管理システムへ入力 |
| 備考 | 帳票を見ながら手入力 |

## 現状の課題

### 記録・追跡に関する課題

| 課題 | 詳細 | 影響 |
|------|------|------|
| **手書き帳票のみ** | デジタル記録がない | 転記ミスのリスク、データ活用困難 |
| **材料消費の追跡不明確** | どのロットの材料を使用したか不明 | 原価追跡・FIFO管理ができない |

### 歩留まり管理に関する課題

| 課題 | 詳細 | 影響 |
|------|------|------|
| **歩留まりの把握不正確** | 投入量と出来高の関係が正確に把握できていない | 加工効率の把握・改善ができない |
| **ロス量の把握不明** | 廃棄部分の記録状況が不明 | 原価計算の精度低下 |

### 原価管理に関する課題（重大）

現在の加工原価はExcelで算出しているが、管理状態に問題がある。

#### 現在の原価計算方法

| 項目 | 内容 |
|------|------|
| 計算ツール | Excel |
| 計算要素 | 出来高、工賃、人数 |
| 工賃の内訳 | 人件費と光熱費を何らかのロジックで按分したもの |

#### 原価管理の課題

| 課題 | 詳細 | 影響 |
|------|------|------|
| **数式の根拠なし** | Excel内に根拠となる数式がない | 計算の正確性・妥当性が検証できない |
| **工賃算出方法が不明** | 人件費・光熱費の按分ロジックが不明 | 原価の信頼性が低い |
| **長期間未更新** | Excelが長らく更新されていない | 現在の実態と乖離している可能性 |
| **属人化** | 計算方法を理解している人がいない可能性 | 担当者変更時に継続不可能 |

### 在庫管理に関する課題

| 課題 | 詳細 | 影響 |
|------|------|------|
| **材料在庫の減算タイミング** | 材料出庫時に在庫が減算されるか不明 | 材料の理論在庫と実在庫の乖離 |
| **製品在庫の増加タイミング** | 出来高入力時に製品在庫が増加するか不明 | 製品の理論在庫と実在庫の乖離 |
